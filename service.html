To implement Odoo, We have two different approaches can be used.

The package approach is an implementation mode that is low cost, generic, with few developments to be done. This approach shall be useful for the small and medium enterprises where most of the applications and report requirement are met by the Odoo Apps.

This custom approach is an implementation mode which is used when the client’s business is weakly covered by Odoo generic. This approach goes from customer’s needs analysis, to Odoo solution.

Odoo Implementation encompasses the whole process of integrating and deploying which includes.
